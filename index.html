Sanetype देवनागरी

Type in the box using English letters. They will get changed to Devnāgrī.
Example:
gMgadhr Hii shk;timan Hai>
turns into
गंगाधर ही शक्तिमान है।
All keybindings

Special keys

q doubles the next letter. Example:
qk → क्क
qkh → क्ख
qsh → श्श
and so on.

qq disables Devnāgrī typing. To re-enable, type qq again.

Pressing a key while holding down alt produces the English letter on that key.

विशेष कुंजियाँ

q से अगले अक्षर का द्वित्व हो जाता है। जैसे:
qk → क्क
qkh → क्ख
qsh → श्श
इत्यादि।

qq से देवनागरी टाइपिंग बंद हो जाती है। वापस चालू करने के लिए फ़िर से qq दबाएँ.

Alt दबाए रखकर किसी कुंजी को दबाने पर उस कुंजी का अंग्रेज़ी अक्षर मिलता है।
