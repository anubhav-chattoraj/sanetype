Sanetype देवनागरी

Sanetype is a Devnāgrī keyboard where most Devnāgrī letters are on the same keys as similar English letters.

Example: The keystrokes
gMgadhr Hii shk;timan Hai>
produce
गंगाधर ही शक्तिमान है।

All keybindings

सेनटाइप एक देवनागरी कुंजीपटल (कीबोर्ड) है जिसमें ज़्यादातर देवनागरी अक्षर मिलते-जुलते अंग्रेज़ी अक्षर वाली कुंजी पर रखे गए हैं।

उदाहरण:
gMgadhr Hii shk;timan Hai>
इन कुंजियों को दबाने से ये वाक्य मिलेगा:
गंगाधर ही शक्तिमान है।

Special keys

q doubles the next letter. Example:
qk → क्क
qkh → क्ख
qsh → श्श
and so on.

qq disables Devnāgrī typing. To re-enable, type qq again.

Pressing a key while holding down alt produces the English letter on that key. This method can also be used to type punctuation marks like ;:'<>

विशेष कुंजियाँ

q से अगले अक्षर का द्वित्व हो जाता है। जैसे:
qk → क्क
qkh → क्ख
qsh → श्श
इत्यादि।

qq से देवनागरी टाइपिंग बंद हो जाती है। वापस चालू करने के लिए फ़िर से qq दबाएँ.

Alt दबाए रखकर किसी कुंजी को दबाने पर उस कुंजी का अंग्रेज़ी अक्षर मिलता है। इसी तरह ;:'<> इत्यादि विराम चिह्न भी टाइप किए जा सकते हैं।
