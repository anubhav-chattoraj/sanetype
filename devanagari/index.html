Sanetype is a Devnāgrī keyboard where most Devnāgrī letters are on the same keys as similar English letters.

Example: The keystrokes
senTaIp Ek devnagrii kiibor;D Hai>
produce
सेनटाइप एक देवनागरी कीबोर्ड है।

(The ; [semicolon] key halves the previous letter.)

All keybindings

सेनटाइप एक देवनागरी कुंजीपटल (कीबोर्ड) है जिसमें ज़्यादातर देवनागरी अक्षर समान ध्वनि वाले अंग्रेज़ी अक्षर की कुंजियों पर रखे गए हैं।

Special keys

Ctrl-M disables Devnāgrī typing. To re-enable, press Ctrl-M again.

Pressing a key after \ (backslash) produces the English letter on that key. This method can also be used to type punctuation marks and Roman digits.
Example:
\a → a   (not ा)
\A → A   (not अ)
\< → <   (not ऱ्‍)
\> → >   (not ।)
\' → '   (not ʼ)
\1 → 1   (not १)
\\ → \
etc.

Pressing Q ignores the surrounding letters.
Example:
AQu → अु   (not औ)
sQh → स्ह   (not श)
etc.

विशेष कुंजियाँ

Ctrl-M से देवनागरी टाइपिंग बंद हो जाती है। वापस शुरु करने के लिए फ़िर से Ctrl-M दबाएँ।

\ (बैकस्लैश) दबाने के बाद कोई कुंजी दबाने से उस कुंजी का अंग्रेज़ी अक्षर मिलता है। इसी तरह विराम-चिह्न और अंगरेज़ी अंक भी टाइप किए जा सकते हैं।
उदाहरण:
\a → a   (ा नहीं)
\A → A   (अ नहीं)
\< → <   (ऱ्‍ नहीं)
\> → >   (। नहीं)
\' → '   (ʼ नहीं)
\1 → 1   (१ नहीं)
\\ → \
वगैरह।

Q दबाने से पिछले अक्षरों का असर हट जाता है।
उदाहरण:
AQu → अु   (औ नहीं)
sQh → स्ह   (श नहीं)
वगैरह।
